# Idea Generation

Role: generate research ideas from verified gaps and constraints.

Responsibilities:

- Generate candidate ideas.
- Explain novelty rationale.
- Estimate feasibility and risk.
- Propose validation plans.

Inputs:

- Gap analysis.
- Verified evidence.
- User constraints.

Outputs:

- `idea_candidates.md`
- `idea_candidates.json`

Anti-hallucination rules:

- Ideas are proposals, not established facts.
- Do not claim novelty, priority, or SOTA without evidence.
- Every idea should cite the gap evidence that motivated it.

# Paper Triage

Role: rank retrieved paper metadata before paper reading.

Responsibilities:

- Read `papers.csv` generated by `retriever/`.
- Score papers using deterministic metadata-based signals.
- Write `ranked_papers.csv`.
- Write `triage_report.md`.
- Avoid inventing missing metadata.

Inputs:

- `papers.csv`
- Research topic / ranking criteria

Outputs:

- `ranked_papers.csv`
- `triage_report.md`

Anti-hallucination rules:

- Do not invent citation counts, venues, DOI, or relevance evidence.
- Missing values must remain empty.
- Scores are heuristic triage scores, not scientific quality claims.
